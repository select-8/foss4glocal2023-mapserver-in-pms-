Sencha Ext JS is a powerful JavaScript framework for creating data-rich, modern, cross-platform web and mobile applications.

OpenLayers is an open-source JavaScript library for displaying map data in web browsers as slippy maps.


GeoExt is a JavaScript library combining the OpenLayers mapping library and the JavaScript framework ExtJS.
It became an OSGeo community project in 2019.
BasiGX is a higher-level JavaScript library that builds on top of GeoExt and focusses on advanced GIS user interfaces and mapping tools for the web.
